# Sway 配置文件结构说明

## 文件组织

这个 Sway 配置被拆分为多个模块化文件，便于管理和维护：

### 主配置文件
- **`config`** - 主配置文件，包含所有其他配置文件的引用

### 模块化配置文件
- **`env.conf`** - 环境变量配置（Qt、SDL、Java 等应用程序的 Wayland 支持）
- **`variables.conf`** - 变量定义（修饰键、方向键、终端、启动器等）
- **`output.conf`** - 显示输出配置（分辨率、壁纸、缩放、间距等）
- **`input.conf`** - 输入设备配置（键盘、鼠标、触摸板设置）
- **`keybindings.conf`** - 按键绑定配置（所有快捷键和组合键）
- **`applications.conf`** - 应用程序启动配置（系统服务、环境变量、剪贴板管理器等）
- **`bar.conf`** - 状态栏配置（Waybar 设置）
- **`utilities.conf`** - 实用工具配置（媒体键、音量、亮度控制）

## 使用方法

1. 所有配置文件都会在主配置文件中自动加载
2. 要修改特定功能，只需编辑对应的配置文件
3. 修改后使用 `$mod+Shift+c` 重新加载配置

## 配置特点

- **双语注释**：每个配置项都有英文和中文注释
- **模块化设计**：功能分离，便于维护
- **完整功能**：包含窗口管理、媒体控制、截图、剪贴板管理等
- **现代化工具**：使用 Waybar、Wofi、Fcitx5、Clash 等现代 Linux 工具

## 主要功能

- 窗口管理和工作区切换
- 媒体播放控制
- 屏幕截图和编辑（Grim + Swappy）
- 剪贴板管理（Cliphist + Wofi）
- 输入法支持（Fcitx5）
- 蓝牙管理（Blueman）
- 代理设置（Clash Verge）
- 系统空闲管理和自动锁屏
- 音量和亮度控制
- 应用程序启动器（Wofi）
